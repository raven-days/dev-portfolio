SKILL

HTML
HTML (Hypertext Markup Language) adalah bahasa dasar yang digunakan untuk membuat struktur halaman web. Saya mahir dalam membuat elemen-elemen seperti heading, paragraph, gambar, dan link untuk membangun website yang terstruktur dengan baik.
CSS
CSS (Cascading Style Sheets) digunakan untuk memberikan gaya visual pada halaman web. Saya memiliki keterampilan dalam styling halaman dengan berbagai teknik, seperti flexbox, grid, dan animasi untuk menciptakan tampilan yang menarik dan responsif.
Javascript
JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif. Saya terbiasa dengan konsep dasar seperti variabel, fungsi, dan manipulasi DOM, serta pengembangan aplikasi web dinamis menggunakan framework dan pustaka JavaScript.
